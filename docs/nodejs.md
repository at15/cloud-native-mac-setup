# Node.js

## Install Node.js

````bash
brew install node@8
````

You need to add it to PATH

````bash
export PATH="/usr/local/opt/node@8/bin:$PATH"
````

## Install cli

- [tldr](https://github.com/tldr-pages/tldr) is man package with example

````bash
npm install -g tldr
tldr tar
npm install -g @vue/cli
````
